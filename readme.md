# Log Archive Tool

 It is a simple tool to archive files such as logs etc. It will help you to keeping your system logs clean and organized while mainatinig the logs for future reference.

 ## Instalation guide

 ### STEP 1
  **Clone repository**
  ```
    git clone https://github.com/xRezek/log_archive_tool
  ```
### STEP 2
  **Add proper premissions**
  ```
    chmod +x log-archive.sh
  ```

### STEP 3
  **Run the tool**
  ```
    log-archive.sh <log-directory>
  ```

  I took the idea from [roadman.sh](https://roadmap.sh/projects/log-archive-tool)
